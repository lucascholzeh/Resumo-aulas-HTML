HTML Semântico

(Voltar)

Nesse primeiro conteúdo, começamos aprendendo sobre o que é e a importância da semântica no HTML.

O que é?

É o uso de tags HTML que deixam claro o significado do conteúdo que elas contêm.

Onde ao contrário de usar div e span para tudo, se organiza com elementos como header, main, footer e etc.

Muito importante para acessibilidade, SEO , manutenção e padronização.

Acessibilidade

(Voltar)

A acessibilidade é um conjunto de práticas que tornam um site utilizável para todas as pessoas, inclusive aquelas com deficiências.

No HTML, isso significa usar legendas ou recursos maiores para que tecnologias possam ler seu site e interpretar para pessoas com dificuldade.

Web Scraping

(Voltar)

Web scraping é uma técnica usada para extrair dados de sites da internet de forma automatizada.

Em vez de copiar e colar manualmente informações de uma página web, o web scraping

usa scripts ou programas para acessar páginas e coletar os dados desejados, como preços, nomes, títulos, links, imagens etc.

Tags main, header e footer

(Voltar)

Essas tags, introduzidas no HTML5, servem para garantir melhor semântica e significado para os conteúdos.

Header

A tag header se usa como cabeçalho para informações no topo da tela.

Main

A tag main serve para colocar o conteúdo principal da sua página e só pode conter UMA dela em cada página.

Pode-se usar as tags header e footer dentro da main.

Footer

A tag footer é usada como rodapé e serve para colocar informações no final da página.

Um bom exemplo são informações e contatos da empresa.

Tags section, aside e nav

(Voltar)

Section

É um elemento semântico que representa uma seção temática de um documento, com conteúdo logicamente relacionado.

A principal diferença entre div e section em HTML está na semântica e no propósito.

Div é um contêiner genérico, usado principalmente para estilização e organização visual.

Ex: Pesquisei sobre casas e vão aparecer várias casas, cada uma dessas vai estar em uma section.

Isso é uma section

Aside

É um assunto que não tem muito a ver com o assunto principal da página, como assunto auxiliares.

Ex: anúncios, biografia do autor, links de blogs, geralmente representadas como barras laterais.

Nav

HTML de navegação, geralmente utilizado junto às listas (ordenadas ou não).

Bom para estilização de partes separadas.

Tags article, blockquote e q

(Voltar)

Article

Destinado para colocar o artigo, o conteúdo principal.

Permite o alinhamento dos textos.

Tem como colocar um article dentro do outro, e o 2o estará relacionado com o 1o que será o “pai”.

Ex: Artigos de notícias, postagens de blog, Comentários de usuários, Entradas de fórum, Widgets interativos.

Blockquote

Usado para citação de um texto, com uma formatação de citação já pré-estabelecida.

Para citar um link, usar o cite=”link” para que o indexador consiga ler e também para melhor acessibilidade.

Texto para exemplo de blockquote

Q

Tambem usado para citação igual o blockquote, só que menor.

Usado para fazer uma citação na mesma linha, sem quebrar ela. Exemplo: Texto para exemplo da tag Q.

Tem como usar o cite também para melhor acessibilidade.

Tags figure, figcaption e picture

(Voltar)

Figure

Como se fosse um container para imagens, onde vai delimitar o espaço delas.

Usado para contextualizar as imagens em meio a um texto.

Figcaption

Tag auxiliar da figure, onde vai colocar a legenda da foto usada no figure.

Usado para relacionar que o texto tem relação com a imagem.

Picture

Também uma tag para imagem, só que nela você pode colocar diversas fontes para uma mesma imagem.

Dá para mudar a imagem se a tela do usuário for menor do que "x" largura.

SEO

(Voltar)

O que é?

SEO (Search Engine Optimization) ou otimização para motores de busca, é um conjunto de técnicas usadas para fazer

com que um site apareça melhor posicionado nos resultados de uma busca no Google, safari etc.

Como funciona?

Os "motores de busca" leem os sites com robôs (crawlers), analisam a estrutura e os conteúdos

e então classificam nos resultados de busca com base em diversos fatores.
